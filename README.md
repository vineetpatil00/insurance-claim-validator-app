# Insurance Claim Validator App

An Angular-based web application for validating and managing insurance claims. This application provides a streamlined interface for processing, reviewing, and validating insurance claims.

## Features

- **Claim Validation**: Create and validate insurance claims with document and image uploads
- **Claims List**: View and manage a list of all insurance claims
- **Modern UI**: Built with Angular and Bootstrap for a responsive, modern interface

## Prerequisites

- Node.js >= 18.0.0
- Angular CLI 18.0.7 or higher
- npm or yarn package manager

## Installation

1. Clone the repository:
```bash
git clone https://github.com/vineetpatil00/insurance-claim-validator-app.git
cd insurance-claim-validator-app
```

2. Install dependencies:
```bash
npm install
```

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Project Structure

- `src/app/claim-validator/` - Main claim validator component
- `src/app/claims-list/` - Claims list component
- `src/app/_services/claims.service.ts` - Claims API service

## Technologies Used

- Angular 18
- TypeScript
- Bootstrap 5 (via CDN)
- RxJS
- Angular Router

## API Configuration

Update the API URL in `src/environments/environment.development.ts` to point to your backend API.

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.dev/tools/cli) page.
